睡眠時無呼吸症候群

睡眠中の呼吸停止による睡眠障害の診断・治療

睡眠時無呼吸症候群とは

睡眠時無呼吸症候群（SAS：Sleep Apnea Syndrome）は、睡眠中に呼吸が停止する睡眠障害です。そのため日中に強い眠気が起き、仕事や日常生活に支障をきたします。

近年、高血圧や脂質異常症、糖尿病といった生活習慣病と関わりがあることが明らかとなりました。当院では睡眠時無呼吸症候群の検査を実施しております。ひどいいびきや日中に眠気の強い方、高血圧がなかなか改善しない方は是非一度簡易検査を受けてみてください。

📊診断基準

  • • 無呼吸：10秒以上の呼吸停止
  • • 一晩に30回以上、または1時間に5回以上
  • • 日本では300〜400万人の患者数と推定
  • • AHI（無呼吸低呼吸指数）5以上で診断

症状

十分な睡眠がとれていない状態ですので、さまざまな症状が表れます。

🌙睡眠時

  • • いびきをかく
  • • 呼吸が止まる
  • • 寝汗をかく
  • • 何度も目が覚める
  • • 頻尿（夜間トイレ）

🌅起床時

  • • 口の渇き
  • • 頭痛
  • • 寝起きが悪い
  • • 体が重い
  • • 疲労感

☀️日中

  • • 強い眠気
  • • 倦怠感
  • • 集中力がない
  • • 記憶力低下
  • • イライラ

原因

喉や上気道が狭くなり、空気が通る十分なスペースがなくなることにより呼吸が止まるのが原因です。

⚖️肥満関連要因

  • • 首・喉まわりの脂肪蓄積
  • • 舌の付け根の気道への落ち込み
  • • 軟口蓋の気道への落ち込み

🫁解剖学的要因

  • • 扁桃肥大
  • • 日本人特有の顎の形
  • • 鼻中隔彎曲症

💡重要なポイント

以前は肥満の方だけに起こると考えられてきましたが、最近では肥満ではなくても睡眠時無呼吸症候群が生じることが分かっています。それは日本人特有の顔、特に顎の形が関係していると考えられています。

生活習慣病との関係

睡眠時無呼吸症候群は肥満、高血圧、脂質異常症、糖尿病などの生活習慣病と密接な関係にあり、様々な合併症を高率に引き起こすことが報告されています。

❤️循環器系への影響

  • • 高血圧
  • • 心筋梗塞
  • • 脳梗塞・脳出血
  • • 心不全
  • • 不整脈

🩸代謝系への影響

  • • 糖尿病
  • • 脂質異常症
  • • メタボリック症候群
  • • 肥満の進行

⚠️社会的リスク

日中の眠気からの居眠り運転や、集中力が欠如することによる交通事故・産業事故により人命が失われたり、社会的な損失を与えているといった報告や報道が多くなされています。

検査方法

問診により睡眠時無呼吸症候群の検査の必要性を認めた場合、簡易型検査装置によるスクリーニング検査を行います。

🏠簡易PSG検査（当院で実施可能）

  • • 自宅で実施
  • • 鼻下と指先にセンサー装着
  • • 呼吸状態と血中酸素濃度測定
  • • AHI測定でスクリーニング

🏥精密PSG検査（連携医療機関）

  • • 入院検査
  • • より精度の高い検査
  • • AHI 40未満の場合に実施
  • • 適切な医療機関をご紹介

📊AHI（無呼吸低呼吸指数）による診断

軽症：AHI 5-15

中等症：AHI 15-30

重症：AHI 30以上

治療

軽症の方は減量や生活習慣の改善だけで症状が良くなることがあります。しかしある程度症状が進んでしまった患者さんは、ひどい眠気のために気力が低下し、減量や生活習慣改善にもなかなか前向きに取り組むことができません。その場合にはCPAP治療の適応となります。

🏃‍♂️生活習慣の改善

  • • 減量
  • • 禁酒・禁煙
  • • 横向き寝
  • • 規則正しい睡眠

🔧その他の治療法

  • • マウスピース療法
  • • 外科的治療
  • • 鼻腔通気度改善

😷CPAP治療（当院で実施可能）

CPAPとは、寝るときに鼻マスクをつけ、空気を送りこむことによって、ある一定の圧力を気道にかけて気道の塞がりを防ぐ治療法です。睡眠時無呼吸症候群のもっとも重要な治療法となっています。

期待される効果

  • • 昼間の眠気改善
  • • 夜間頻尿改善
  • • 頭痛の改善
  • • 睡眠の質向上

予防効果

  • • 高血圧の改善
  • • 心筋梗塞の予防
  • • 脳卒中の予防
  • • 心不全の予防

当院での対応

当院では、睡眠時無呼吸症候群の診断から治療まで、患者様一人ひとりの状態に応じた包括的な医療を提供しています。
